Instruction for room assembly:

  • Starting from a corner of the room, connect them using a top corner connection, then continue by building out the sides, connecting using the top pieces. Either assemble the walls upside down, or flip them once assembled.
  • Now upside-down, place the floor tiles on top, connecting them to the walls as you go. Once fully assembled, flip it back over, using a large board for support if need be.
  • Note: Inner corners use a four-way connection instead of a corner bottom.
  • Plug in all the pieces, and run the website, importing the python file provided here.
