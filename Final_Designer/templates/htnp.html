Use this page to create the puzzle flows.
Left clicking the area to the right will create a node on a grid near your mouse.
Right clicking the node or connection will remove it.
Double clicking the node allows you to rename it.

Each node can take an input from the room and/or produce an output back to it. You can link nodes together to have one room input trigger multiple events, or require multiple inputs to trigger one event. You can connect nodes by left clicking the input box (to the left of the node) of one and the output box (to the right of the node) of another.
